## 端口配置

| 服务 | 端口 | 说明 |
|------|------|------|
| 后端 API | 3001 | 主要业务接口 |
| Web 前端 | 3000 | Next.js 开发服务器 |
| Nginx | 80/443 | 生产环境反向代理 |
| PostgreSQL | 5432 | 数据库 |

## API 调用流程

### APK 端
```
APK → HTTP → http://43.129.16.148:3001/api/* → 后端
```

### Web 端 (开发环境)
```
浏览器 → HTTP → http://localhost:3000 → Next.js → http://localhost:3001/api/* → 后端
```

### Web 端 (生产环境)
```
浏览器 → HTTPS → baizhiji.net → Nginx → 127.0.0.1:3001/api/* → 后端
```

## 数据流

1. **用户认证**
   - 用户登录 → 验证 → 返回 JWT Token
   - Token 存储在 localStorage (Web) 或 SecureStorage (APK)

2. **API 请求**
   - 所有请求携带 `Authorization: Bearer <token>`
   - 后端验证 Token 后返回数据

3. **AI 生成**
   - 用户请求 → 后端 → 阿里云/腾讯云 API → 返回结果
   - 结果存入素材库

4. **内容发布**
   - 用户选择素材 → 选择账号 → 后端调用 Playwright → 发布到平台

## 环境变量

```env

JWT_SECRET=your-jwt-secret
PORT=3001
NODE_ENV=production

# AI 服务
DASHSCOPE_API_KEY=your-aliyun-key
TENCENT_SECRET_ID=your-tencent-id
TENCENT_SECRET_KEY=your-tencent-key
```

### Web (.env.local)
```env
NEXT_PUBLIC_API_BASE_URL=http://43.129.16.148:3001/api
```

### APK (api.config.ts)
```typescript
BASE_URL: 'http://43.129.16.148:3001/api'
```
