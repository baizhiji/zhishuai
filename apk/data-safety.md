# Google Play 数据安全表单

## 数据收集与安全

### 收集的数据类型

| 数据类型 | 是否收集 | 用途 | 是否加密 | 用户是否可删除 |
|---------|---------|------|---------|-------------|
| 姓名 | 是 | 账户管理 | 是 | 是 |
| 手机号 | 是 | 账户认证、短信通知 | 是 | 是 |
| 邮箱 | 是（可选） | 账户恢复 | 是 | 是 |
| 用户生成内容（AI生成、上传素材） | 是 | 服务核心功能 | 是 | 是 |
| 设备ID | 否 | - | - | - |
| 精确位置 | 否 | - | - | - |
| 粗略位置 | 否 | - | - | - |
| 应用使用数据 | 是 | 服务优化 | 是 | 是 |
| 崩溃日志 | 是 | Bug修复 | 是 | 否（匿名） |
| 性能数据 | 是 | 性能优化 | 是 | 否（匿名） |

### 数据安全措施
- 传输加密：HTTPS/TLS
- 存储加密：AES-256
- 访问控制：基于角色的权限管理

### 第三方SDK
- 无第三方数据分析SDK
- 支付通过微信支付官方SDK
- 地图服务使用腾讯地图SDK（可选）
