# Debug / 调试脚本归档

本目录存放开发过程中产生的临时调试、检查、修复脚本。

这些脚本大多是为解决特定问题一次性编写的，保留在此以备参考。
**生产环境请使用 `scripts/` 和 `deploy/` 下的正式脚本。**

## 文件分类

| 类别 | 前缀 | 数量 | 用途 |
|------|------|------|------|
| 检查脚本 | `check_*.py` | ~12 | 检查 API 路由、SSL 状态、服务器配置等 |
| 调试脚本 | `debug_*.py` | ~2 | 调试认证、错误等 |
| 验证脚本 | `verify_*.py` | ~6 | 验证功能、部署状态等 |
| 测试脚本 | `test_*.py` | ~3 | 测试 AI 对话、路由等 |
| 修复脚本 | `fix_*.py` | ~25 | 各种问题修复（API/SSL/部署/密码等） |
| 最终验证 | `final_*.py` | ~4 | 上线前的最终验证 |
| FK 脚本 | `fk_*.py` | ~8 | FK 相关操作 |
| 其他 | - | 2 | full_check.py, force_run.py |

## 注意事项

- 这些脚本可能包含硬编码路径或临时方案
- 部分脚本可能有重复功能（多次迭代版本）
- 如需重新运行，建议先检查脚本中的配置是否仍适用
