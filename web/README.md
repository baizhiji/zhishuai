# 智枢AI SaaS系统 - Web端管理后台

## 项目简介

智枢AI是一个全功能商业智能平台，集成自媒体、电商、HR、营销、获客等全场景功能。Web端管理后台提供完整的配置和管理能力。

## 技术栈

- **框架**: Next.js 14.2.0 (App Router)
- **语言**: TypeScript 5.4
- **UI组件库**: Ant Design 6.3.6
- **状态管理**: Zustand 4.5
- **日期处理**: Dayjs 1.11
- **HTTP客户端**: Axios 1.7
- **样式**: Tailwind CSS 3.4

## 功能模块

### 1. 自媒体板块 ⭐⭐⭐⭐⭐
- **AI内容生成**: 基于热点话题和行业特点生成视频、图文、短视频内容
- **矩阵账号管理**: 管理抖音、快手、小红书、视频号等多平台账号
- **批量发布**: 一键上传到各平台，自动填写标题、描述和标签
- **数据统计**: 查看播放量、点赞、评论、分享等数据，导出报表

### 2. 电商板块 ⭐⭐⭐⭐⭐
- **智能详情页生成**: 基于产品信息生成吸引人的商品详情页
- **多店铺管理**: 管理淘宝、京东、拼多多、抖店、美团等店铺
- **自动上架**: 一键上架到各平台，自动填写商品信息
- **价格监控**: 实时监控竞品价格，提供调价建议
- **销量统计**: 查看销量、转化率、客单价等关键指标

### 3. HR功能 ⭐⭐⭐⭐
- **职位发布**: 自动发布到BOSS直聘、前程无忧、智联招聘、拉勾网
- **AI简历筛选**: 自动接收简历，AI分析匹配度
- **自动回复**: 自动回复候选人消息
- **面试安排**: 管理面试日程和面试反馈

### 4. 获客功能 ⭐⭐⭐⭐⭐
- **潜在客户发现**: 根据关键词、话题发现潜在客户
- **自动发送信息**: 自动发送引流信息
- **二维码发送**: 自动发送二维码并追踪效果
- **转化统计**: 查看发送记录和转化率

### 5. 推荐分享 ⭐⭐⭐⭐
- **二维码生成**: 生成专属推广二维码
- **推荐链接生成**: 生成专属推荐链接
- **推荐追踪**: 实时查看推荐数据

### 6. 转介绍 ⭐⭐⭐⭐
- **我的推荐**: 查看推荐的用户列表和推荐状态

### 7. 账号管理 ⭐⭐⭐
- **用户管理**: 管理用户账号
- **代理商管理**: 管理区域代理商
- **客户管理**: 管理终端客户

### 8. 系统配置 ⭐⭐⭐
- **API配置**: 配置阿里云百炼、火山引擎等服务商
- **知识库管理**: 上传和管理知识库文件
- **APP定制**: 配置APP名称、Logo、主题等

## 安装和运行

### 安装依赖

```bash
cd zhishuai-web
npm install
```

### 开发模式

```bash
npm run dev
```

访问 http://localhost:3000

### 构建生产版本

```bash
npm run build
npm start
```

### 类型检查

```bash
npm run type-check
```

### 代码检查

```bash
npm run lint
```

## 环境变量

创建 `.env.local` 文件：

```env
APP_NAME=智枢AI
APP_VERSION=1.0.0
API_BASE_URL=http://localhost:3000/api
```

## 账号体系

系统支持三级账号：

1. **Admin（开发者总后台）**: 最高权限，管理所有账号和配置
2. **Agent（区域代理）**: 管理终端客户，开通客户账号
3. **Customer（终端客户）**: 使用APP功能，后台可以使用所有功能

## 页面导航规则

- Dashboard（首页）→ 板块页面 → 功能操作页面
- 功能页面返回到板块页面
- 板块页面返回到Dashboard

## 开发规范

- 所有页面使用中文
- 所有页面支持返回上级
- 使用TypeScript严格模式
- 遵循Ant Design设计规范
- 代码提交前运行类型检查

## API集成

系统采用"中转站"架构，依赖第三方API：

- **主力平台**: 阿里云百炼（95%场景）
- **备用平台**: 火山引擎（高并发场景）

### 阿里云百炼模型

| 类别 | 模型 | 用途 |
|------|------|------|
| 文本类 | qwen3.6-plus | 商品文案、内容创作、客服对话 |
| 图像类 | wan2.7-image-pro | 商品图、封面图、海报生成 |
| 视频类 | wan2.7-t2v | 文字生成视频、营销视频 |
| 数字人类 | wan2.7-digital-human-clone | 数字人形象克隆 |
